## 使用方法

### 单按钮版本

如果你的硬件上面只有一个按钮,那么软件上你只能使用单按钮版本的代码,具体如何切换烧录请见Code

对于Button1,短按一下是模式一*(甩干)*,再按一下就是模式二*(快速洗)*,再按一下就是模式三*(标准洗)*,再按一下就回到模式一;

短按到模式一时蜂鸣器会短响一次,选中模式二时蜂鸣器会连续短响两次,选中模式三时蜂鸣器会连续短响三次;

对于Button1,长按会启动发送数据,此时蜂鸣器会先发出长叫声,再短响一次,此时开始工作.

### 双按钮版本

如果你的硬件上有两个按钮,那么你既可以使用单按钮版本软件代码,也可以使用双按钮版本软件代码

对于Button1,短按一下是模式一*(甩干)*,再按一下就是模式二*(快速洗)*,再按一下就是模式三*(标准洗)*,再按一下就回到模式一;

短按到模式一时蜂鸣器会短响一次,选中模式二时蜂鸣器会连续短响两次,选中模式三时蜂鸣器会连续短响三次;

对于Button2,长按会启动发送数据,此时蜂鸣器会先发出长叫声,再短响一次,此时开始工作.

(个人推荐单按钮版本,节约一个按钮,少一点硬件却能做相同的事情,但是双按钮版本确实可以有效地解决误触的风险)

## 简介

V1.0是在特定环境下产生的特定版本.它的优势是体积小,成本低,约为3元以下,器件少;供电+5V;有两个按钮位置,可用根据实际情况焊接,建议只使用按钮1即可
